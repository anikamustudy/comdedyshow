# Comedy Show Booking Website

Full-stack MERN app for booking comedy shows with QR tickets, Razorpay payments, admin panel.

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env  # Add your keys
npm run dev
```

### Frontend  
```bash
cd frontend
npm install
npm run dev
```

### Features
- Dynamic show listing with video previews
- Seat booking + UPI/Card payments (Razorpay)
- QR ticket generation & verification
- Admin dashboard
- User auth (JWT)

**Backend runs on http://localhost:5000**  
**Frontend runs on http://localhost:5173**
